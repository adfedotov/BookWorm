BookWorm

BookWorm is a small personal project to interact with python's Flask web framework

All of the books (covers, titles, authors, etc.) are provided by OpenLibrary API

Technologies used: Python (Flask), HTML, CSS, jQuery

This is an ongoing project that I will keep improving during my free time. Things that need to be improved:

  • Handling of errors when OpenLibrary doesn't provide certain book information
  • Completely redesign the website front-end with SASS
  • Consider storing ISBN together with OLID in database if for some reason there will be an unexpected change in their system
  • Connect SMTP service to send out registration confirmation and news
  • More on GitHub page.
